# CyCTF-LearnStream
this repo contain the scripts for solving learnstream challenge from CyCTF 2023


# HOW TO USE
1. ```python3 server.py```    -> will listen on `http://127.0.0.1:8000`
2. ```python3 solv.py http://127.0.0.1:8000/video.mp4```
